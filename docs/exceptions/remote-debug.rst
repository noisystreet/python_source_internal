调试支持 — pdb 与底层调试协议
====================================

Python 的调试器（``pdb`` ）基于 ``sys.settrace`` 实现。这一节简要介绍
调试器的底层工作方式。

从一道题开始
------------

``pdb`` 的核心机制：

- 设置 ``sys.settrace(trace_dispatch)``
- 在 ``trace_dispatch`` 中检查当前行是否包含断点
- 如果命中断点，进入交互式命令循环

第一问：断点管理
----------------

断点信息存储在 ``sys`` 模块的 ``breakpointhook`` 中。

第二问：性能影响
----------------

启用追踪后，每条字节码执行前都会调用追踪函数——性能下降显著。
